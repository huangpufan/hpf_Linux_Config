# TUI Installer 快速入门

## 30 秒上手

```bash
# 1. 进入目录
cd tui-installer

# 2. 安装依赖
pip install -e .

# 3. 运行
python -m tui_installer
```

## 基本操作

启动后你会看到一个分类列表和工具列表。

### 导航

- **方向键** 或 **hjkl**: 移动光标
  - `h`/`←`: 上一个分类
  - `l`/`→`: 下一个分类
  - `j`/`↓`: 下一个工具
  - `k`/`↑`: 上一个工具

### 选择与安装

1. 用 **Space** 勾选要安装的工具
2. 按 **a** 批量安装所有已勾选的工具
3. 或者按 **i** 只安装当前高亮的工具

### 查看日志

- 按 **Enter** 切换到日志视图，查看安装过程
- 再按 **Enter** 返回列表视图

### 退出

按 **q** 退出程序。

## 使用技巧

### 1. 预先获取 sudo 权限

很多安装脚本需要 sudo 权限。建议在运行前执行：

```bash
sudo -v
```

这样可以避免安装过程中卡住等待密码输入。

### 2. 并发安装

选择多个工具后按 `a`，它们会同时并发安装，大大节省时间。

### 3. 查看实时日志

安装过程中，选中正在运行的任务，按 `Enter` 可以查看实时输出日志。

## 常见问题

### Q: 安装卡住不动了？

A: 可能是 sudo 会话过期了。按 `Ctrl+C` 退出，运行 `sudo -v` 后重试。

### Q: 提示 "需要SSH密钥配置，已跳过"？

A: 某些工具需要访问 GitHub 私有仓库。请先配置 SSH 密钥：

```bash
ssh-keygen -t ed25519
cat ~/.ssh/id_ed25519.pub
# 将公钥添加到 GitHub Settings -> SSH Keys
```

### Q: 如何添加自己的安装脚本？

A: 编辑 `tui_installer/data/tools_config.json`，按照现有格式添加条目即可。
